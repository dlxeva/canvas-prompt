# Whiteboard Example 001

This folder is reserved for the first public example.

Target contents:

- a final canvas image
- a minimal Prompt Package JSON
- a short human explanation of what was drawn
- optionally a short transcript

## Intended purpose

The example should show that a whiteboard session is not just an image.

It is a structured thinking trace that can be interpreted by AI through:

- final canvas semantics
- drawing order
- emphasis
- optional voice explanation
